# Dodge the Creeps

Videojuego desarrollado siguiendo la documentación oficial de Godot y su apartado "[Tu primer juego 2D](https://docs.godotengine.org/es/4.4/getting_started/first_2d_game/index.html)".



Con este proyecto pude comenar a entender el lenguaje de programación GDScript y el funcionamiento general del motor gráfico.
